# Users

## What is a User?

A **User** is a professional (CA, lawyer, consultant) who uses SnapDocs.

Users:
- Own all data
- Create clients
- Create requests
- Track document progress

---

## Why Users Exist

- Clear data ownership
- Enables multi-user orgs later
- Prevents data leakage
